HOMEWORK

WARM UP (DUE IN DRIVE FOLDER BY 9AM NEXT WEEK)

You may have forgotten some programming concepts over the break. Let's see what remains... For next week, create 2 small Processing sketches which include a mix of:

  1. A custom class that you created.
  2. Mouse input
  3. Keyboard input
  4. An image or video
  5. An if-else-else if conditional statement
  6. A for() loop
  7. Some form of movement

Remember, Processing has great online reference materials if you've forgotten how to do any of these things.

Also, looking at old classwork is totally fine, but I would recommend not simply remixing past work into this homework.

This isn't a test. I want you to recognize and be able to communicate your weakspots.
